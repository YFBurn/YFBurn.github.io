物尽其用，士思其行

发表于 2017-03-07 | 分类于 随笔 | | 本文总阅读量 次

前两天刚看完日本作家山下英子的作品《断舍离》，是一本关于物资整理的书，里面包含着些许处事、抉择的原理。整本书断断续续地读了两周，才阅读记录完毕，内容里没有具有冲击力的内容，可让人为之一叹。前半部分的内容，让我感觉拖沓，不爽快，有点像是凑字数，同样的观点反复提及，后半部分倒是有几分味道。

01 何为“断舍离”

以下引引自作者山下英子的原话：

“物品要用才有价值——是为断。”
“物品要在此时、当下，应出现在需要它的地方——是为舍。”
“物品处于恰当的位置，才能展现美感——是为离。”

“断”，顾名思义，截断。控制物品进入我们生活的数量，只购买对我们有使用价值的物品。
“舍”，丢弃。丢弃不需要、不合适、不舒服的物品。
“断”与“舍”相辅相成，最终达成“离”的状态——“通过有意识的选择，让物品自然而然地回归它应该在的地方，回归到需要它的地方。”

02 物尽其用

通读本书，我觉得这四个字可以概括作者书中的“断舍离”提及的基本理念。

物

在现在这个社会当中，物质泛滥，众物之中，是“物”必有竞者，或两者取其一，抑或是百里挑一、万里寻一。
此“物”并不是要最为珍贵、最为精致。
“物”的基本准则是：需要、合适、舒服。
在准则之上，我们还要加以抉择，才能得到最满意的、最称心的物品。
在喜欢的物品中挑出最喜欢的几件，比如5件、3件，加以区分、甄别，控制同类、类似功能物品的总数量（山下英子的“七、五、一原则”），反复进行“断舍离”，最后保留下来的物品就是自己最喜欢、最经常使用的。
在挑选物品的时候，我个人认为，可以尝试购买品味、档次稍微高于自己收入状况的（并不是指高于收入总额）。
我始终相信，一分钱一分货，免费的东西往往是最贵的。比如说，付费的设计，表面上看起来和免费的差别不大，但是住户进去后所有的细节体验是完全不一样的。
便宜的东西不一定不好，贵的东西也不一定好用。当然，谈这些极端的例子是没有意义的。
去年，我的大学同学刘开煌来福州，和我一起考BIM，考试结束后们就出去玩了。那两天时间里，我们吃了两家很不错的饭店：大叔食堂、藏福。
两家店味道真的是很不错，但是价格也会贵一些。自从那次以后，我才慢慢意识到，穷人和富人接触的世界，差别将会是多么的巨大。
所以，能讲究的时候，不瞎将就。在踮起脚尖能够触及的情况下，给自己更好的一个选择，不仅仅是物质享受，也是自我的一个激励，奋斗的动力。

尽

拥有的物品要能够得到充分的利用，尘封起来的东西，和扔掉了没有什么区别，反而还要花费精力去清理、整理。要让物品得到充分的利用，那么在挑选“物”的过程中，就不能马虎，不能随意。将就的东西，留与不留也变得将就。

其

“其”即“它”。物与人，客与主，被用与用。关系其实很简单。
但是，有些时候，人往往会忘记了自己立场。
—“哎呀，这个扔了多可惜啊，还能用（吃）。”
—“还能用”而不是“我要用”。
扔掉后就会造成浪费，而浪费的根源就是瞎购买酿下的罪因。
反客为主，就容易造成当“断”不“断”。

用

倘若将物置换为人，以对人的态度来对待事物。
被闲置的物品，就比如一个喜欢你的小姑娘，而你对她置之不理，它每天要目睹你跟着其他姑娘（如果你喜欢汉子我也没话说）干点什么事情，这种滋味，参考马冬梅吧！
但是既然你不喜欢，不愿意，那么就饶了人家黄花大闺女吧，该走的都让他们走。
人与人的关系是这样，人与物的关系也是类似。
闲置的物品，无法发挥价值，留着不就占据空间，反而还要花心思去整理，得不偿失。

03 士思其行

书中讲的道理都是从整理房间的物品中提炼出来的，道理不仅仅是说物品，同时也会影射到人生。
整理好自己的房间，还自己一个焕然全新的心情。
我的宿舍是公司分配的，一般是一个房间2个人住，今年我的舍友因为外区域工作需要，被调到外区域去了，现在剩下我一个人住一个房间。
我们这个房间，面积大约有15m2，是一个阳台改造成的房间，两个人住显得十分拥挤。上周，我对宿舍的格局进行重新分配，把堵在窗口的两个衣柜挪到入门左侧的墙边，虽然路线变窄了，但是现在我每天早上起来都能看到窗外明媚阳光，小房间也一改阴霾的面貌。

书中还提到了一个比较有意思的词语，“可惜”。
这让我想起当代作家王思成的作品《别扯了，时间才不会改变一切》中的一个例子：
作者偶然间获得一家高档自助餐的优惠券，带着平时省吃俭用的父母去品尝品尝。等到最后吃饱后，店家又送了一个大冰激凌过来，作为饭后甜点，大家都吃不下了，作者便去结账。回来后发现，母亲却已经把冰激凌吃了一大半，母亲觉得不吃太可惜了。
母亲单纯觉得冰激凌不吃了太可惜了，而并不是说母亲自己想吃，即便自己已经饱了，也要将冰激凌消灭。
而作者也觉得很可惜，母亲为了一个冰激凌而强行进食，万一身体吃出问题那不就得不偿失。
可惜，有两种含义。往往抉择不同，产生的结果也大相径庭。
比如说，你在A地，领导通知你去B地开会，你开车开到一半的路程，领导突然再次通知会议取消。那么你现在就面临两个选择：
1、好可惜啊，都到一半了，那我继续去B地吧。
2、掉头回去。
相信不同的人，会有不同的选择。选择 1 的人，被已经消耗的时间和路程给主导了，觉得不继续下去很可惜；选择 2 的人知道自己去了B地已经没有任何意义了，去了也是浪费时间，很可惜。
在人生中也有一些这样的例子，从业3年5年，发现自己越来越不喜欢自己的工作，是继续从业还是选择换岗？这毕竟不是路程的问题，需要考虑的因素很多，每个人针对自身的情况，得出的结论才是最合适的，才不会日后一直觉得很可惜。

“断舍离”从整理杂物的过程中，提炼出为人处世的方法，难能可贵，文中的需要细小的观点值得慢慢体会。
整理自己的生活、工作，就像整理物品一样，需要“断舍离”。
物品通过“断舍离”可以达到物尽其用，希望通过“断舍离”我们能够明确自己的行动方向。

为梦想努力！
