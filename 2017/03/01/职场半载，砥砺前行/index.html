职场半载，砥砺前行

发表于 2017-03-01 | 分类于 随笔 | | 本文总阅读量 次

01
步入职场已经半年有余了，期间体会了不少酸甜苦辣，每每想起其中的辛苦，总会笑自己当时脑子进水，去读了工程管理，从此成为一名搬砖界的标杆。
半年多的时间了，除了工作，更多的时间是在迷茫，很多次地询问自己，我喜欢这份工作吗？走上这条路以后，我是一辈子都和工地打交道了吗？
刚从大学毕业出来的我，内心无比渴望自由，财务自由、人身自由、精神自由。但是现实从来都不会让每一个天真的人得逞。
拿着每个月微薄的工资，看着日益上涨的房价，比工资涨得还快的快餐费，我发现我现在已经学会了对比：沃尔玛购物的价格的对比、同事之间工资的对比、同事之间生活状态的对比等等。我发现，大学四年出来，我就成了廉价的劳动力，没有生产力也没有积蓄。

02
物与物的差异，人与人的差异，差异存在生活中的每一个角落。
差异而产生了不平等。是的，人生而不平等，上帝都不可能把一碗水端平。
在公司做事，就算你做的事情是最脏、最累、最吃力的活，你的得到回报肯定不会是最高的。在一个公司中，充满着各种的评判标准，有些事情上甚至连标准都没有，这就会导致最累的人得不到期望的回报，一旦期望得不到满足，就会产生失望。
不单单是自己失望，连旁边的人看到也会失望！久而久之，便没有人愿意冲锋陷阵，没有人愿意赴汤蹈火。

03
你认为事情不应该是这样的。但是事实真的是这样，也应该是这样的。
付出最多劳动力的，承担的风险最少，回报就会少。就比如基金的收益是比不上股票的。但是这都只是一般的规律，生活往往有趣得多，经常让你摔个狗吃屎。
劳动，是工作上的常态；而思考，才是突破自己当前局限的唯一武器。
常言道，八小时以内求生存，八小时以外求发展。
我用了很长的时间去思考自己的工作和自己内心的需求。
我很庆幸，我已经认识到了，我想要什么，我也正在尝试在做我感兴趣的事情，也在努力让自己能力变强。
半年了，我一直都没有喜欢上福州这个城市。
半年了，这份工作曾经让我开心过，也曾经让我迷茫过。不过，现在的我，正如陈师傅所言，白纸上的任何一笔，都是收获。感谢这份工作，感谢这个平台所提供给我的每一个机会。
半年了，我也该静下心来好好安排自己好好规划自己。

为梦想努力！
