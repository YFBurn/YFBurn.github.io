尝试使用ITFFF

发表于 2017-09-01 | | 本文总阅读量 次

起因

最近几天我的室友谢晋一直在专研ONE ONTE的使用方法，他自己已经使用这个软件很久了，一直在跟我说这个软件的牛逼程度。

之前大学快毕业的时候，把自己的小破笔记本升级到了WIN 10，电脑中便有了ONE NOTE这个软件，当时我也觉得挺好用的，毕竟可以做到电脑与手机同步。（嗯，当时就是这么没有出息，就只有这个要求。）
但是后来使用的频率少了，在现场干活根本不怎么需要用到电脑，有个破台式机能发个OA流程及发个联系单就可以。

但是谢晋玩的就666了，他给我展示他的ONE NOTE里面的各种笔记（内容区分），内容检索，画笔功能，复制自动生成网页链接等等。可以说，他已经离不开这个软件了。
就在他昨天继续钻研的这个软件的时候，他发现了一个周边程序——IFTTT.

什么是IFTTT

ifttt是“if this then that”的缩写，事实上是让你的网络行为能够引发连锁反应、让你使用更为方便，其宗旨是“Put the internet to work for you”（让互联网为你服务）。

简而言之的话，就是做一个命令，让2个程序相互关联起来，让其遇到设定的情况时，执行相应的操作。

因吹斯听！

IF weibo then Email

今天中午就和谢晋在尝试微博与Email之间的命令，其实挺简单的，但是因为英语比较蹩脚，搞了好久才搞懂。
刚开始让他先视奸我的微博，我发微博测试。

有文化真可怕！

后来我自己也尝试了一下，发现上手并不是特别难（主要还是英语难），后面还关联了一个天气插件，每天定时给我发第二天的天气预报。


IF github then Day One

其实写这篇文章的目的是在测试，在hexo端写的博客是否能够利用IFTTT的功能同步到Day One，毕竟在电脑端无法使用Day One。

为梦想努力！
